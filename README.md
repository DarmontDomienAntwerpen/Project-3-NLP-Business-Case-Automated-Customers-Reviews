## Setup

### 1. Download the dataset
Download the 3 CSV files from Kaggle and place them in `data/raw/`:

рџ‘‰ [Amazon Product Reviews вЂ” Kaggle](https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products)

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Install Ollama (for summarization вЂ” notebook 04)
```bash
brew install ollama
ollama pull llama3.2
```

### 4. Run notebooks in order
```
01 в†’ 02 в†’ 02b в†’ 03 в†’ 04
```
This generates all processed files in `data/processed/` and the fine-tuned model in `data/finetuned_model/`.

### 5. Start Ollama server (keep this terminal open)
```bash
ollama serve
```

### 6. Run the web app
```bash
streamlit run app.py
```
