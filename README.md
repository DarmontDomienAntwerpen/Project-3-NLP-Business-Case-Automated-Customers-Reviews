# Amazon Review Analyzer
**IronHack Project — Domien Darmont**

An end-to-end NLP pipeline that classifies, clusters, and summarizes Amazon product reviews — deployed as an interactive Streamlit web app.

---

## Pipeline

```
Raw Reviews (46,169)
    ↓
01 Preprocessing      → reviews_clean.csv
    ↓
02 Classification     → Fine-tuned BERT (accuracy: 79.4%)
    ↓
03 Clustering         → KMeans + all-mpnet embeddings (k=5)
    ↓
04 Summarization      → Llama3.2 3B via Ollama
    ↓
App                   → Streamlit web app
```

## Setup

### 1. Download the dataset
Download the 3 CSV files from Kaggle and place them in `data/raw/`:

👉 [Amazon Product Reviews — Kaggle](https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products)

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Install Ollama (for summarization — notebook 04)
```bash
brew install ollama
ollama pull llama3.2
```

### 4. Run notebooks in order
```
01 → 02 → 02b → 03 → 04
```
This generates all processed files in `data/processed/` and the fine-tuned model in `data/finetuned_model/`.

### 5. Start Ollama server (keep this terminal open)
```bash
ollama serve
```

### 6. Run the web app
```bash
streamlit run app.py
```

## Models

| Component | Model | Performance |
|-----------|-------|-------------|
| Classification | `nlptown/bert-base-multilingual-uncased-sentiment` (fine-tuned) | Accuracy: 79.4% |
| Clustering | `all-mpnet-base-v2` + KMeans (k=5) | Silhouette: 0.343 |
| Summarization | `llama3.2` (3B) via Ollama | ROUGE-1: 0.276 avg |
